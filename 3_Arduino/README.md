## ЁЯУН Pin Connections

### Encoders
```
Encoder X:
  - Pin A: GPIO 13
  - Pin B: GPIO 12

Encoder Y:
  - Pin A: GPIO 14
  - Pin B: GPIO 27
```

### Motor X (L298N #1)
```
Motor Driver:
  - IN1: GPIO 26
  - IN2: GPIO 25
  - ENA: GPIO 18 (PWM)

Motor:
  - OUT1, OUT2 тЖТ Motor X
```

### Motor Y (L298N #2)
```
Motor Driver:
  - IN3: GPIO 16
  - IN4: GPIO 17
  - ENB: GPIO 19 (PWM)

Motor:
  - OUT3, OUT4 тЖТ Motor Y
```

### Water Pump (Relay)
```
Relay Module:
  - Signal: GPIO 21
  - VCC: 5V
  - GND: GND

Pump:
  - + тЖТ Relay NO (Normally Open)
  - - тЖТ GND
```
